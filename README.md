# Learn Computer Graphics

* 使用 `C` 語言
* OS: Windows 10
* 計算機圖學的學習筆記，持續更新中...

## 使用工具
* gcc 8.1.0
* cmake 3.11.1

## 如何建置
* 請先確認是否安裝`cmake`(3.1以上)與`gcc`，並設置於環境變數。
* 開啟`build.bat`進行建置
